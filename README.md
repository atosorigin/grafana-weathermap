# WeatherMap

WeatherMap Panel Plugin for Grafana

Compatibility with  Prometheus


# Installation

Weathermap requires [Grafana](https://www.grafana.com/) v6.6+ to run (not tested lower version)
Download and install it

## Compatibility

- Grafana 6.6.x
- Yarn 1.21.1
