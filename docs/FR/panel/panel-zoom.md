# Zoom

[![](../../screenshots/other/Go-back.png)](README.md)

![menu](../../screenshots/panel/plusmoins.png)

Après avoir sélectionné le bouton `Zoom [P / M]`, il est possible d'agrandir le rendu en tapant sur le clavier l'une des touches suivantes : 

 - `P` : pour grossir
 - `M` : pour réduire


Chaque objet conservera la proportion définie

Lorsque le viseur ne nécessite pas d'agrandissement ou de contraction, il est nécessaire de cliquer à nouveau sur le bouton pour lui redonner sa couleur d'origine
